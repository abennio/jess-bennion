# Jess Bennion Portfolio

React portfolio website for Jess Bennion's design work, built with Vite and deployed under `/jess-bennion/` on GitHub Pages.

## Available Scripts

### `npm start`

Runs the Vite development server at [http://localhost:3100](http://localhost:3100).

### `npm test`

Runs the test suite once with Vitest.

### `npm run test:watch`

Runs Vitest in watch mode for local development.

### `npm run build`

Builds the production site into the `dist` folder.

### `npm run preview`

Serves the production build locally at [http://localhost:4173](http://localhost:4173).
